Web client
========

### Prerequisite
- [Node.js] 6.0+
- [yarn] *or* [npm]

### Intructions
```bash
yarn          # Install all dependencies
yarn build    # Build everything in production mode

yarn watch    # Run webpack in watch mode
```

[Node.js]: https://nodejs.org/en/
[yarn]: https://yarnpkg.com
[npm]: https://www.npmjs.com
